Application review checklist guide

Use this guide to help you complete the application review checklist.

Eligibility

Check the applicant:

  • meets the eligibility requirements as set out on GOV.UK
  • is applying for £35,000 to £500,000 of funding
  • will use the water for irrigation

If the applicant is not eligible, you’ll need to recommend the application for rejection.

Documents

Applicants must have uploaded all of the following:

  • abstraction licence
  • 3 years of business accounts or other financial evidence
  • new and existing irrigation system plans and drawings
  • other licences, permissions and consents
  • planning permission and approved plans
  • project match funding evidence

If applicable, they also need to have uploaded:

  • a landlord underwriting form
  • supplier confirmation for second-hand funding
  • a tenancy agreement
  • VAT evidence

If any documents are missing or incorrect, contact the applicant.

Costs, suppliers and quotes

Check all the project items are eligible as set out on GOV.UK, and that quotes have been supplied for all items.

Dual funding

Check if the applicant has applied to fund any project item using the following grants:

  • Countryside Stewardship (CS)
  • Farming in Protected Landscapes

If the applicant has already applied for funding using another grant, contact the applicant to confirm which grant they want to use to fund the item.

Due diligence checks

Use the bankruptcy and insolvency register and disqualified directors register to check if any directors have:

  • been disqualified
  • been declared insolvent
  • gone bankrupt

You should also use the get information about a company service to check the business is still trading.
